LabMind AI

Real-time AI-powered Lab Safety Monitoring System

  Features
- Live camera detection using YOLOv8
- Safety alerts (Goggles, Gloves detection)
- Modular architecture (CV, Backend, Frontend)
- check the safety and then give the instruction 
 
 Tech Stack
- Python
- YOLOv8 (Ultralytics)
- OpenCV

Problem
Labs lack real-time supervision leading to unsafe practices.

Solution
LabMind AI detects unsafe behavior and alerts instantly.

  Project Structure
- cv/ → computer vision logic
- backend/ → API and processing
- frontend/ → user interface
